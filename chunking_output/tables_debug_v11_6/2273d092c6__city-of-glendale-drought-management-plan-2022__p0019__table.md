|  |  | SECTION 5. BEST MANAGEMENT PRACTICES |  |  |  |  |  |
|  |  |  |  | The City may use several best management practices to deal with droughts. These practices |  |  |  |
|  |  |  | include both supply management and demand management strategies. |  |  |  |  |
| SUPPLY MANAGEMENT |  |  |  |  |  |  |  |
|  |  |  |  | The City's first line of defense in facing drought is to fully use all water resources that are |  |  |  |
|  |  |  |  | available to replace affected water supplies. Supply management strategies are generally |  |  |  |
|  |  |  |  | preferred over demand management strategies because they have minimal impact on water |  |  |  |
|  |  |  |  | users. In many cases, the public may not even realize that the City has implemented supply |  |  |  |
|  |  | management measures to deal with drought.. |  |  |  |  |  |
|  |  |  |  | Glendale is fortunate because it is not reliant on a single source of water. It has several options |  |  |  |
|  |  |  |  | to provide water during drought, including surface water and groundwater. If drought causes a |  |  |  |
|  |  |  |  | cutback in the City's surface water supply, the City will consider using all existing wells and non- |  |  |  |
|  |  |  |  | impacted surface water that can produce additional supply prior to asking water users to curb |  |  |  |
|  |  |  |  | water consumption. City wells would be used to access the City's groundwater and |  |  |  |
|  |  |  |  | underground stored water credits. The City will apply to ADwR for a drought groundwater |  |  |  |
|  | pumping exemption if necessary. |  |  |  |  |  |  |
|  | DEMAND MANAGEMENT |  |  |  |  |  |  |
|  |  |  |  | The second level of response when faced with drought is reducing water demand by |  |  |  |
|  |  |  |  | implementing water reduction measures. Each measure can be designed to match the level of |  |  |  |
|  |  |  |  | severity of water shortage and the type of water user. Generally, the implementation of |  |  |  |
|  |  |  |  | measures will begin with a public information and drought awareness communications plan |  |  |  |
|  |  |  |  | encouraging voluntary water use reductions. As the drought impact becomes more severe, |  |  |  |
|  |  |  | mandatory water reduction measures may be implemented. |  |  |  |  |
|  |  |  |  | Water reduction measures focus primarily on outdoor water use because water savings can be |  |  |  |
|  |  |  |  | achieved without compromising public health and safety. Outdoor water use mainly affects |  |  |  |
|  |  |  |  | aesthetics, while indoor water use directly affects hygiene and sanitary conditions. |  |  |  |
