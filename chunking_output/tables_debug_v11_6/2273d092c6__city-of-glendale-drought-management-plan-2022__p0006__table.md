| DROUGHT MANAGEMENT |  |  |  |  |  |  |  |
|  |  |  |  | While the terms drought management and water conservation are sometimes used |  |  |  |
|  |  |  |  | interchangeably, there is a notable and important difference. Drought management refers to |  |  |  |
|  |  |  |  | the plans and actions a city or water provider will implement, usually temporarily, to reduce the |  |  |  |
|  |  |  |  | impact of drought and water shortage on its water system. This includes switching to |  |  |  |
|  |  |  |  | alternative water supplies and implementing water reduction measures to save water. |  |  |  |
| WATER CONSERVATION |  |  |  |  |  |  |  |
|  |  |  |  | Water conservation is long-term and designed to reduce the amount of water used regardless |  |  |  |
|  |  |  |  | of drought. As a desert community, it is especially important to be water wise. The City |  |  |  |
|  |  |  |  | developed a water conservation public education program in 1985 in response to the 1980 |  |  |  |
|  |  |  |  | Groundwater Management Act, which required water providers in the state's Active |  |  |  |
|  |  |  | Management Areas (AMA) to create water conservation programs. |  |  |  |  |
|  |  |  |  | Water conservation serves an important role in helping Glendale provide reliable water |  |  |  |
|  |  |  |  | supplies for the future and enabling the City to stay in compliance with the state water |  |  |  |
|  |  |  |  | conservation requirements. Glendale maintains over ten water conservation programs listed as |  |  |  |
|  |  |  |  | Best Management Practices (BMPs) by the Arizona Department of Water Resources (ADwR). |  |  |  |
|  |  |  |  | These programs are designed to promote ongoing water conservation behaviors and practices |  |  |  |
|  | through information, education, and incentives. |  |  |  |  |  |  |
|  |  |  |  | Water conservation is an integral part of Glendale's long-term water management strategy by |  |  |  |
|  |  |  |  | helping reduce the demand for water. From 2000 to 2020, Glendale's population increased by |  |  |  |
|  |  | 6%, yet the water demand decreased by 10% (Figure 1). |  |  |  |  |  |
