|  |  |  |  | Scottsdale was among the first communities in the nation to adopt several important transportation |  |
|  |  |  |  | policies: creating a comprehensive citywide trails system, accommodating bicycle users through |  |
|  |  |  |  | comprehensive planning and facilities, and adopting the safer lagging left-turn arrows to manage |  |
|  |  |  |  | intersection traffic. Highlights of current transportation programming include the free Scottsdale Trolley, |  |
|  |  |  |  | a gold-level Bicycle Friendly Community designation, over 270 miles of paths and trails, and a state-of- |  |
|  |  |  |  | the-art traffic management system to reduce congestion and emissions. Trip reduction surveys show |  |
|  |  |  |  | that the largest municipal locations have seen a steady 5-year reduction in single-occupancy vehicle |  |
|  |  |  |  | commuting (82% down to 66%, adjusted for telecommuting) and that Scottsdale is the single biggest user |  |
|  |  | of the Maricopa County vanpool. |  |  |  |
